Écris ton prénom : plateforme éducative pour les jeunes enfants afin d'apprendre à écrire leur prénom !

## Installation

1. installer et démarrer le backend : `rails s `
1. `npm install`
1. `npm start`
1. Go to [http://localhost:3001](http://localhost:3001)
1. Lezgo !
