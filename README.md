## Pitch du projet

Écris ton prénom : plateforme d'apprentissage pour jeunes enfants, pour apprendre à écrire son prénom !

## Installation

1. `bundle install`
1. `npm install`
1. `npm start`
1. Go to [http://localhost:3000](http://localhost:3000)
1. Lezgo !
